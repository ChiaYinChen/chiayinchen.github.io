使用 Jupyter Notebook 運行 R 語言環境

 Posted on December 29, 2017  |  1 min (173 mots)

推薦安裝 Anaconda，因為安裝 Anaconda 的同時也已安裝 Jupyter Notebook。安裝完 Anaconda 以後，可以在程式集裡看到 Anaconda Navigator、Anaconda Prompt、IPython、Jupyter Notebook、spyder 等。

以下以 Anaconda 3 在 Windows 上的操作為例：

首先，打開 Anaconda Prompt 的視窗，接下來在當前的環境中安裝 R Essentials ：

# Anaconda Prompt
conda install -c r r-essentials

出現 Proceed <[y]/n> 後，按下 Enter ！

完成以後，就已經創建好 R Essentials 環境囉，同時也安裝了資料科學常用的 R 套件，包含 dplyr、shiny、ggplot2、tidyr、caret 和 nnet 等套件，可以使用 conda list 查看在此環境下安裝的套件。

啟動 Jupyter Notebook 查看 R 環境是否已經整合到 Jupyter：

# Anaconda Prompt
jupyter notebook

運行 jupyter notebook 指令後，會在瀏覽器中啟動 Jupyter Notebook 介面。


趕緊來試一下 R 語言在 Jupyter 上的效果：

  • 檢視資料

  • 使用 dplyr 套件整理資料 

library(dplyr)

iris %>%
  group_by(Species) %>%
  summarise(mean_Sepal.Width = mean(Sepal.Width)) %>%
  arrange(desc(mean_Sepal.Width))

  • 使用 ggplot2 套件繪圖 

library(ggplot2)

ggplot(data = iris, aes(x = Sepal.Length, y = Sepal.Width)) +
  geom_point(aes(color = Species, shape = Species)) +
  ggtitle("Sepal.Length vs. Sepal.Width") +
  xlab("Sepal Length") +
  ylab("Sepal Width") +
  theme(plot.title = element_text(size = 15, hjust = 0.5))
